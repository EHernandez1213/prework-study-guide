HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is the structure of a webpage.
  • Child elements are elements nested inside a body element.
  • h1-h6 are elements representing headings and decrease in size as number goes up.
  • p elements represent paragraphs and block texts.
  • ul, ol, and li represent unordered lists, ordered lists, and list items.
  • img represents elements containing images. The src attribute being the location of the image.
  • br creates lines of empty space.
  • a element is an anchor which creates hyperlinks. href is the attribute for the url.
